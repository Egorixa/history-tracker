Импорт истории браузера из JSON

Загрузите файл с экспортом истории (формат расширений типа «Chrome History Export»). Принимаются: массив объектов с полем url (и опционально title), либо массив строк-URL.

Применяются ваши текущие белый и чёрный списки из popup. Дата каждого визита на сервере = «сейчас» — без bulk-эндпоинта историческое время сохранить нельзя.

Каналы назначения

Пересечение канала с импортированной историей

Загрузите JSON истории и выберите канал — посчитаем три множества: что есть и там и там, что есть только в истории, что есть только в канале. Сравнение по нормализованному URL.

Без bulk-эндпоинта приходится постранично выкачать визиты канала (по 200 за запрос). Для крупных каналов — медленно.
